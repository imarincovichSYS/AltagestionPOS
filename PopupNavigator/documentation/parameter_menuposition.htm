MenuPosition Parameter

Enables you to display menu to the right of the button.

Format

VALUE="<MenuPos>"
DEFAULT="BOTTOM"

<MenuPos> can have one of the following values:

  • BOTTOM - Menu is displayed in the bottom of the button.
  • RIGHT - Menu is displayed to the right of the button.

Example

Display menu to the right.
