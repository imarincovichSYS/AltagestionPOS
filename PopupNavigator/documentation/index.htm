Popup Navigator Applet v1.1.1

Overview

PopupNavigator is Java applet for web-site navigation. It is intended to be used by webmasters who want to add a flexible navigation tool to their web sites without consuming significant screen space or download time. Applet looks like a button that, when pressed, activates a hierarchical popup menu. Items of the menu represent other pages. User can click on the menu item to display appropriate page.

I wrote this applet for my own site, but since there is obvious lack of small and functional navigational applets, I decided to make it publicly available. This applet is completely freeware and full source code is included. You may freely redistribute it as long as it is unchanged. If you want to change it, you must obtain my permission.

When used in JAR archive (recommended method), it is less then 11 KB.

If you like it and use it on you web site, I would appreciate if you could include a link to Branko's Enthusiastic Programming Pages (http://bepp.8m.com) or a credit somewhere within your site or PopupNavigator itself, but this is not obligatory.

Compatibility

Every JDK v1.1.3 compatible browser will run it. This includes IE4 or newer and N4.5 or newer. The previous version had a problem with Macintosh compatibility. In this version I made changes that should improve Mac compatibility, but since I don't have access to any Mac box, I was unable to test this. I would appreciate any feedback on this issue.
