DelimiterChar Parameter

This parameter specifies the delimiter char for sub-parameters and sub-menu items.

Format

VALUE="<Char>"
DEFAULT=";"

Example

Let's say we want to use "," as delimiter instead of ";".

PopupNavigator Java Applet - JDK 1.1.3 needed

<applet
  width="128"
  height="24"
  code="PopupNavigator/PopupNavigatorApplet.class"
  codebase="./"
  archive="PopupNavigator.jar">
<param name="DelimiterChar" value=",">
<param name="0" value="BEPP,http://bepp.8m.com">
<param name="1" value="Submenu">
<param name="1,0" value="Test Page 1,test1.htm">
<param name="1,1" value="Test Page 2,test2.htm">
PopupNavigator Java Applet - JDK 1.1.3 needed
</applet>
