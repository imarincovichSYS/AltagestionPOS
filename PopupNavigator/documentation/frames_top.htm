"top" frame

We have a frameset with two frames: "top" and "bottom". PopupNavigator is placed in the top frame, but target pages should be displayed in the bottom frame. We can use DefaultFrame to easily redirect all URLs without explicit frame specification to the bottom frame. There are also couple of URLs with explicitly specified frames - as a demonstration.

PopupNavigator Java Applet - JDK 1.1.3 needed

<applet
  width="128"
  height="24"
  code="PopupNavigator/PopupNavigatorApplet.class"
  codebase="./"
  archive="PopupNavigator.jar">
<param name="DefaultFrame" value="bottom">
<param name="0" value="BEPP;http://bepp.8m.com">
<param name="1" value="Test Page 1;test1.htm">
<param name="2" value="Test Page 2;test2.htm">
<param name="3" value="Explicit Frames">
<param name="3;0" value="Test Page 1, _self;test1.htm;_self">
<param name="3;1" value="Test Page 2, _parent;test1.htm;_parent">
<param name="3;2" value="Test Page 3, _top;test1.htm;_top">
<param name="3;3" value="Test Page 4, _blank;test1.htm;_blank">
<param name="3;4" value="Test Page 4, bottom;test4.htm;bottom">
PopupNavigator Java Applet - JDK 1.1.3 needed
</applet>
