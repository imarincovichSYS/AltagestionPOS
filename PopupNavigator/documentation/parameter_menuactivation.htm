MenuActivation Parameter

Use this parameter to set event that activates (displays) popup menu.

Format

VALUE="<MenuActivationMethod>"
DEFAULT="CLICK"

<MenuActivationMethod> can have one of the following values:

  • CLICK - menu is activated when user clicks on the button.
  • MOUSEOVER - menu is activated when user places mouse pointer over the button.

Example

Menu is activated immediately when user places mouse pointer over the button.
