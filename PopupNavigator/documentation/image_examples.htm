Image Examples

These examples illustrate use of Image and ImagePosition parameters.

Example #1

Display a JPEG image as a background. Since ImagePosition is not used, image left upper corner is placed in the left upper corner of the button.

PopupNavigator Java Applet - JDK 1.1.3 needed

<applet
  width="96"
  height="96"
  code="PopupNavigator/PopupNavigatorApplet.class"
  codebase="./"
  archive="PopupNavigator.jar">
<param name="Image" value="clouds.jpg">
<param name="0" value="BEPP;http://bepp.8m.com">
<param name="1" value="Test Page 1;test1.htm">
<param name="2" value="Test Page 2;test2.htm">
PopupNavigator Java Applet - JDK 1.1.3 needed
</applet>

Example #2

Display a GIF image in the top of the button label.

PopupNavigator Java Applet - JDK 1.1.3 needed

  height="64"
  code="PopupNavigator/PopupNavigatorApplet.class"
  codebase="./"
  archive="PopupNavigator.jar">

<param name="ImagePosition" value="TOP;4">
<param name="0" value="BEPP;http://bepp.8m.com">
<param name="1" value="Test Page 1;test1.htm">
<param name="2" value="Test Page 2;test2.htm">
PopupNavigator Java Applet - JDK 1.1.3 needed
</applet>
